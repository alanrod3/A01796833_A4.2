# Pruebas y Calidad de Software - Actividad 4.2

Este repositorio contiene la entrega de la Actividad 4.2, enfocada en el desarrollo de herramientas estadísticas y de procesamiento de datos siguiendo estándares de calidad de software (**PEP8**), manejo de errores y eficiencia en la ejecución.

##  Estructura del Repositorio

El proyecto está organizado por ejercicios (P1, P2, P3), cada uno con su propio entorno de pruebas y resultados:

```text
Pruebas_y_Calidad/
└── 4.2/
    ├── P1/ (Compute Statistics)
    │   ├── source/ -> computeStatistics.py
    │   ├── tests/  -> Archivos de datos (TC1 - TC7)
    │   └── results/ -> StatisticsResults.txt y evidencias
    ├── P2/ (Convert Numbers)
    │   ├── source/ -> convertNumbers.py
    │   ├── tests/  -> Archivos de datos (TC1 - TC4)
    │   └── results/ -> ConvertionResults.txt y evidencias
    └── P3/ (Word Count)
        ├── source/ -> wordCount.py
        ├── tests/  -> Archivos de datos (TC1 - TC5)
        └── results/ -> WordCountResults.txt y evidencias
